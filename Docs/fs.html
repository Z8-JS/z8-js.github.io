File System API

Z8 provides a highly optimized, Node-compatible file system module available via node:fs and node:fs/promises.

Z8 Advantage: The fs module utilizes Adaptive Buffering for synchronous operations and a Task-based Thread Pool for asynchronous operations, resulting in performance up to 15x faster than competitors.

Synchronous API

Access these methods via import { ... } from 'node:fs'.

readFileSync(path: string, encoding?: string): string | Uint8Array

Reads the entire contents of a file. Returns a Uint8Array by default, or a string if 'utf8' encoding is specified.

writeFileSync(path: string, data: string | TypedArray): void

Writes data to a file, replacing the file if it already exists.

appendFileSync(path: string, data: string | TypedArray): void

Appends data to a file. Highly Optimized: Z8 uses a Persistent Handle strategy for high-frequency appends, avoiding expensive open/close syscalls.

Asynchronous (Promise) API

Access these methods via import { ... } from 'node:fs/promises' or node:fs.

readFile(path: string, encoding?: string): Promise<string | Uint8Array>

Asynchronously reads the entire contents of a file. Returns a Uint8Array or string via the Z8 Job System.

writeFile(path: string, data: string | TypedArray): Promise<void>

Asynchronously writes data to a file. Optimized for mass-concurrency.

Other Synchronous Methods

statSync(path: string): Stats

Returns a Stats object containing file information (size, mtime, atime, etc) and methods like isFile(), isDirectory().

mkdirSync(path: string, options?: { recursive: boolean }): void

Synchronously creates a directory. Supports recursive creation.

rmSync(path: string, options?: { recursive: boolean }): void

Removes files and directories. Supports recursive deletion.

existsSync(path: string): boolean

Returns true if the path exists, false otherwise.

renameSync(oldPath: string, newPath: string): void

Renames or moves a file/directory.

copyFileSync(src: string, dest: string): void

Synchronously copies a file.

realpathSync(path: string): string

Returns the resolved absolute path.
